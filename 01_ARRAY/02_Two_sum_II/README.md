# Two Sum II - Input Array Is Sorted

**LeetCode:** 167  
**Difficulty:** Medium

## Problem
Given a sorted array of integers `numbers` and a target, return the indices of the two numbers whose sum equals the target.

## Approach
- Use two pointers: one at the beginning and one at the end.
- Compare their sum with the target.
- Move the left or right pointer accordingly until the pair is found.

## Time Complexity
- **O(n)**

## Space Complexity
- **O(1)**
