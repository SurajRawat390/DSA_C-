# LeetCode 167 - Two Sum II

## Approach
- Since the array is sorted, use the **Two Pointer** technique.
- Start one pointer from the beginning and one from the end.
- If the sum is equal to the target, return the indices.
- If the sum is smaller, move the left pointer.
- If the sum is larger, move the right pointer.

## Time Complexity
- **O(n)**

## Space Complexity
- **O(1)**

## Concepts
- Arrays
- Two Pointers
